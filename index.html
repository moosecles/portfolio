Toxicity Classifier

My Toxicity Classifier consists of two versions, each utilizing a different algorithm - Multinomial Naive Bayes and LSTM RNN. This project was developed using Python, pandas, TensorFlow, Keras, NumPy, scikit-learn, and NLTK to determine sentence toxicity and assess its degree of toxicity for various applications.

Robo-Moose

"Robo-Moose" is a versatile Discord music bot, adaptable to various servers through user-friendly setup instructions. Equipped with music commands like play, skip, and pause, it utilizes Lavalink for smooth music playback, Hikari-py for organized code, and asynchronous functions for seamless user interactions.

One-Spoiler

"OneSpoiler" is a Python Reddit bot fetching weekly One Piece manga spoilers via the Reddit API and PRAW module. It features a Tkinter GUI for spoiler browsing, stores the last viewed chapter, and prioritizes organized code with PRAW. Upcoming improvements include chapter switching and displaying the latest chapter for reading.
