Robo-Moose
"Robo-Moose" is a fully functional Discord music bot that can be easily customized for use in different servers. By following the provided instructions, users can set up the bot, handle server hosting, and input their own Discord and bot information. The bot offers a range of music-related commands, including playing, skipping, stopping, pausing, resuming, seeking, showing the queue, shuffling, and more. The implementation includes Lavalink for seamless music playback, Hikari-py for efficient code structure, and asynchronous functions for responsive bot interactions.
One-Spoiler
This project entails creating a Python Reddit bot called "OneSpoiler" that fetches the latest spoilers for the One Piece manga series every Thursday. The bot utilizes the Reddit API and the PRAW module to retrieve chapter spoilers from the corresponding subreddit. It maintains a record of the last seen chapter and its organized Tkinter-based GUI allows users to browse and view the spoiler text conveniently. The project emphasizes clear code structure by implementing PRAW and follows the API's guidelines. Future development aims to enhance the bot's functionality by adding features like switching between chapters and providing the latest chapter for reading.
